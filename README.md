# Dev-waste_classifier
built a CNN-based image classification model to classify waste into \textbf{4 categories}: Biodegradable, Non-Biodegradable, Recyclable, and Hazardous.

\item Collected and preprocessed a dataset of \textbf{5,000+ images}, including resizing, normalization, augmentation, and train-validation-test splitting.

\item Trained the model for \textbf{50 epochs} using TensorFlow, achieving \textbf{92.4\% training accuracy} and \textbf{89.1\% validation accuracy}.

\item Improved model generalization through image augmentation, reducing overfitting and increasing validation accuracy by approximately \textbf{8\%}.

\item Integrated OpenCV for real-time image capture and inference, producing predictions in under \textbf{300 ms} per image on a local system.
